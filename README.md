# 🌸 Smart Flower Lens

An AI-powered web application that identifies flowers from uploaded images. Built with Python, Streamlit, and the power of Google's Gemini Flash AI model.

## ✨ Features
* **Accurate Identification:** Upload an image of any flower, and the AI will identify it.
* **Multiple Flowers:** Can detect and identify multiple different flowers in a single image.
* **Bilingual Output:** Provides detailed botanical and medicinal information in both **English** and **Hindi**.
* **Clean UI:** Simple, interactive, and easy-to-use interface built on Streamlit.

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **AI Model:** Google Gemini (generativeai)
* **Image Processing:** Pillow (PIL)

## 🚀 How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/piyush9026-del/smart-flower-lens.git
2. Install the required libraries:
   ```bash
   pip install -r requirements.txt
3. Add your Gemini API Key in a .env file:
   ```bash
   API_KEY="your_api_key_here"
4. Run the application:
   ```bash
   streamlit run Flowerapp.py
